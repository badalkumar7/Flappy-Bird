<h1>Flappy-Bird</h1>
<p>This is a Flappy Bird game developed using Java and Java Swing. The project is inspired by the classic Flappy Bird arcade game and demonstrates core programming concepts such as game loops, event handling, collision detection, gravity mechanics, and score tracking. The player controls a bird that must navigate through gaps between moving pipes without hitting them. This project helped strengthen my understanding of object-oriented programming and real-time game development in Java.
</p>
